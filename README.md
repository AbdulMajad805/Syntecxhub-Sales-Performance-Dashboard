# 📊 Sales Performance Dashboard – Power BI Project

## Overview

This project focuses on building an interactive **Sales Performance Dashboard** using Power BI. The dashboard was created to analyze sales data and present meaningful business insights through clean and interactive visualizations.

The main goal of this project was to understand how business data can be transformed into useful reports that support better decision-making.

---

# Project Goals

* Analyze overall sales performance
* Track monthly sales trends
* Compare sales across regions and product categories
* Identify top-performing products
* Build an interactive and user-friendly dashboard

---

# Tools Used

* Power BI
* Power Query
* DAX (Basic Calculations)
* Microsoft Bing Maps
* Kaggle Dataset

# Dataset Information

Dataset source:

Sales Forecasting Dataset from Kaggle

---

# Data Preparation

Before creating the dashboard, the dataset was cleaned and transformed in Power Query.

The following steps were performed:

* Removed duplicate records
* Checked and handled null values
* Corrected column data types
* Fixed date formatting issues
* Created additional columns for:

  * Month Name
  * Month Number
  * Year
  * Shipping Days

---

# Dashboard Contents

## KPI Metrics

The dashboard includes:

* Total Sales
* Total Orders
* Total Customers
* Average Sales

## Visual Reports

* Monthly Sales Trend
* Sales by Region
* Sales by Category
* Sales by Segment
* Top 10 Products
* State-wise Sales Map

## Interactive Filters

Users can filter dashboard data using:

* Year
* Region
* Category

---

# Key Insights

* The West region recorded the highest sales performance.
* Technology products contributed a major share of revenue.
* Consumer customers generated significant sales activity.
* Sales patterns varied across different months.
